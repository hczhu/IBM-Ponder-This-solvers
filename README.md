# IBM Ponder This Solvers

My source code for solving some [IBM Ponder This](https://research.ibm.com/haifa/ponderthis/index.shtml) puzzles.

## Build and Run

### C++ Solvers
To build all C++ solvers:
```bash
make
```

To run tests:
```bash
make test
```

### Python Solvers
To run Python solvers (e.g., November 2025):
```bash
python3 2025-11.py solve
```
To run Python tests:
```bash
python3 2025-11.py
```
